# HerSaathi Store Release Checklist

## Before Internal Testing

```text
[ ] Google Play Developer account ready
[ ] App created in Play Console
[ ] Package name com.hersaathi.app confirmed
[ ] Official email her.saathi.care@gmail.com added
[ ] App category selected
[ ] Store listing draft reviewed
[ ] App icon uploaded
[ ] Feature graphic created
[ ] Phone screenshots captured
[ ] Privacy Policy URL hosted and working
[ ] Terms URL hosted and working
[ ] Data deletion page hosted and working
[ ] Internal tester emails added
[ ] AAB built with production profile
[ ] AAB uploaded to internal testing
```

## Before Closed Testing

```text
[ ] Billing/deferred services decision confirmed
[ ] Firebase Functions deployed if cloud AI/sync will be tested
[ ] Gemini secret deployed if Saathi cloud AI will be tested
[ ] OpenAI secret deployed if Pragya will be tested
[ ] Firestore rules deployed
[ ] Google sign-in tested on APK
[ ] Cloud upload/download tested
[ ] Data Safety form completed
[ ] Content rating completed
[ ] Target audience completed
[ ] Privacy Policy matches actual behavior
[ ] Legal copy reviewed
```

## Before Production

```text
[ ] All critical/high tester issues fixed
[ ] Version code increased
[ ] Production AAB built
[ ] Payment/subscription system connected or Premium hidden
[ ] Premium entitlement backend connected
[ ] Account/cloud deletion workflow confirmed
[ ] Public policy URLs final
[ ] Store screenshots final
[ ] Data Safety final
[ ] App signed and uploaded
[ ] Production rollout plan selected
```

## Current Blockers For Public Launch

```text
[ ] Billing not enabled
[ ] Cloud Functions not deployed live
[ ] Gemini/OpenAI secrets not deployed live
[ ] Real subscription/payment provider not connected
[ ] Public Privacy Policy URL not hosted yet
[ ] Public Terms URL not hosted yet
[ ] Final legal review pending
```
