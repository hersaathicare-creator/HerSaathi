HerSaathi Data Deletion

Last updated: May 28, 2026

Delete Local Data

Open HerSaathi, go to Profile, open Data Management, and choose Delete All Local Data. This clears local records on that device.

Delete Cloud Backup

If signed in, open Profile and use Delete Cloud Backup to delete the HerSaathi Firestore backup for your Google account.

Request Account Or Cloud Deletion

Email her.saathi.care@gmail.com with the subject "HerSaathi data deletion request". Include your signed-in email and Firebase UID if available in the app.

Processing Time

Deletion requests should be reviewed and processed as soon as reasonably possible after support can verify the request.
